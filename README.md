## ▶️ How to Run Locally

Clone the repository:

```bash
git clone https://github.com/Fadboy31/devjobs.git
```

Navigate into the project folder:

```bash
cd devjobs
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open in browser:

```bash
http://localhost:5173
```
